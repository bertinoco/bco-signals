BCO Skills Tracker — Settings

GitHub Configuration

Create a token with repo scope at github.com/settings/tokens
Format: owner/repo-name

How to Create a GitHub Token

  1. Visit github.com/settings/tokens
  2. Click "Generate new token" → "Generate new token (classic)"
  3. Give it a name: "BCO Skills Tracker Extension"
  4. Select scopes: repo (full control of private repositories)
  5. Click "Generate token" and copy it
  6. Paste it into the "GitHub Personal Access Token" field above
